✨ 项目特色

这是一个完全运行在浏览器端的轻量级数独游戏，无需后端支持，无需构建工具，只有一个 .html 文件。

• 🎨 极致像素风：采用 VT323 字体，配合复古配色与 CSS 像素阴影，致敬 Game Boy 时代。


• 🔊 8-bit 合成音效：使用 Web Audio API 实时合成音效（填入、擦除、胜利音乐），无需加载任何 MP3 资源文件。

• 📱 全平台适配：针对 iPhone 和 iPad 进行了触控优化，完美支持移动端与桌面端。

• 🖨️ 智能打印模式：独家功能！点击打印按钮，自动生成 9 个互不相同 的数独题目，并以 3x3 布局 完美适配 A4 纸张，适合制作线下练习册。

• 🧠 完整游戏逻辑：
• 标准数独生成算法（保证唯一解）。

• 3 种难度选择 (Easy, Medium, Hard)。

• 笔记模式 (Pencil Marks)。

• 实时错误检查与计时器。

• ⚡ 单文件架构：所有逻辑、样式、字体引用集成在一个 index.html 中，部署极其简单。

🚀 快速开始

本地运行
1. 下载本项目中的 index.html 文件。
2. 直接使用 Chrome / Safari / Edge 浏览器打开该文件。
3. 开始游戏！

部署到 GitHub Pages (推荐)

想要分享给朋友玩？只需 3 步：
1. 在 GitHub 上创建一个新仓库（例如 pixel-sudoku）。
2. 将 index.html 文件上传到该仓库。
3. 进入仓库 Settings -> Pages，将 Branch 设置为 main 并保存。
4. 等待 1 分钟，点击生成的链接即可在线游玩。

🎮 操作说明
• 填入数字：点击棋盘格子，再点击下方的数字键盘。
• 笔记模式：点击 ✎ NOTES 按钮切换模式，可以在格子里标记多个候选数字。
• 擦除：点击 ⌫ ERASE 或键盘 Backsapce 删除数字。
• 打印：点击 🖨️ PRINT 9 按钮，浏览器将弹出打印预览，自动生成一页 A4 纸 9 个题目。

🛠️ 技术栈
• HTML5 / JavaScript (ES6+)
• Tailwind CSS (CDN) - 快速样式构建
• Web Audio API - 音频合成
• CSS Grid - 响应式布局与打印布局
